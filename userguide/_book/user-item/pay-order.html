发起支付

请求地址： pingpp/pingppOrder

HTTP请求方式： POST

请求参数 类型 是否必选 参数说明
bookingId long true 订单id
quick boolean false 加急
money double false 金额
time String false 预约时间
type String false 支付类型

注：

time ： yyyy-MM-dd HH:mm

type ：1支付宝，2微信，3银联
